## CSS

| Global class               | Description                                          |
| -------------------------- | ---------------------------------------------------- |
| .ods-token-input           | Styles applied to the input element.                 |
| .ods-token-input\_\_input  | Styles applied to the input element.                 |
| .ods-token-input--error    | Styles applied to the input element if error={true}. |
| .ods-token-input--disabled | Styles applied to the element if is disabled         |
