# TopBar API

The API documentation of the TopBar React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Input } from '@useblu/ocean-react';
```

## Usage

## Props

<ArgsTable of="." />

The ref is forwarded to the root element.

Any other props supplied will be provided to the root element (native element).

## CSS

| Global class          | Description                                              |
| --------------------- | -------------------------------------------------------- |
| .ods-TopBar           | Styles applied to the TopBar element.                    |
| .ods-TopBar--filled   | Styles applied to the TopBar element when it's filled.   |
| .ods-TopBar--disabled | Styles applied to the TopBar element when it's disabled. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/TopBar/TopBar.tsx) for more detail.

## Inheritance

The props of the [FormControl](?path=/docs/components-formcontrol--page) component are also available.
