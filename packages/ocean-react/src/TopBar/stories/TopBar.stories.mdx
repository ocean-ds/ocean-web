<ArgsTable of="." />

## CSS

| Global class           | Description                                                |
| ---------------------- | ---------------------------------------------------------- |
| .ods-topbar            | Styles applied to the TopBar element.                      |
| .ods-topbar-extend     | Styles applied to the TopBar element extend variant.       |
| .ods-topbar-title      | Styles applied to the TopBar element when it's title.      |
| .ods-topbar-search     | Styles applied to the TopBar element when it's search.     |
| .ods-topbar-scroll-bar | Styles applied to the TopBar element when it's box-shadow. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/TopBar/TopBar.tsx) for more detail.
