# IconButton API

The API documentation of the IconButton React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { IconButton } from '@useblu/ocean-react';
```

## Usage

<Canvas withSource="open" withToolbar>
  <Story name="Usage">

  </Story>
</Canvas>

## Props

<ArgsTable story="Usage" of="." />

The ref is forwarded to the root element.

Any other props supplied will be provided to the root element (native element).

## CSS

| Global class            | Description                         |
| ----------------------- | ----------------------------------- |
| .ods-icon-btn           | Styles applied to the root element. |
| .ods-icon-btn--[sm,md]  | Specifies the width of the button.  |
| .ods-icon-btn--disabled | Styles of a disabled button.        |
