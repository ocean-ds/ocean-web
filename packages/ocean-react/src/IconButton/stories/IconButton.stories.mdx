The API documentation of the IconButton React component. Learn more about the props and the CSS customization points.


## CSS

| Global class           | Description                         |
| ---------------------- | ----------------------------------- |
| .ods-iconbtn           | Styles applied to the root element. |
| .ods-iconbtn--[sm,md]  | Specifies the width of the button.  |
| .ods-iconbtn--disabled | Styles of a disabled button.        |
