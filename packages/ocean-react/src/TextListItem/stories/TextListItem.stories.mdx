<ArgsTable of="." />

## CSS

| Global class                               | Description                                 |
| ------------------------------------------ | ------------------------------------------- |
| .ods-text-list-item                        | Styles applied to the root element.         |
| .ods-text-list-item--with-action           | Styles applied if with action.              |
| .ods-text-list-item--wide                  | Styles applied if element is wide           |
| .ods-text-list-item--selectable            | Styles applied if has checkbox or radio.    |
| .ods-text-list-item\_\_default-content     | Styles applied to the default text content. |
| .ods-text-list-item\_\_info-text           | Styles applied to the info text.            |
| .ods-text-list-item\_\_info-text--neutral  | Styles applied if info text type neutral.   |
| .ods-text-list-item\_\_info-text--positive | Styles applied if info text type positive.  |
| .ods-text-list-item-selectable             | Styles applied if is selectable.            |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/TextListItem) for more detail.
