# TextArea API

The API documentation of the TextArea React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { TextArea } from '@useblu/ocean-react';
```

## CSS

| Global class          | Description                                              |
| --------------------- | -------------------------------------------------------- |
| .ods-textarea         | Styles applied to the textarea element.                  |
| .ods-textarea--filled | Styles applied to the textarea element when it's filled. |
| .ods-textarea--error  | Styles applied to the textarea element if error={true}.  |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/TextArea/TextArea.tsx) for more detail.
