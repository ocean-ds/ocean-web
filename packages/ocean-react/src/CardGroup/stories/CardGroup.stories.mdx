# CardGroup

The `CardGroup` component is used to group related information in an organized layout. It supports different variants (`minimal` and `header`) and can include a title, subtitle, counter, and a clickable action.

## Global Classes

Below are the global classes applied to the `CardGroup` component:

| Global Class                       | Description                                                       |
| ---------------------------------- | ----------------------------------------------------------------- |
| `.ods-card-group`                  | Styles applied to the root element of the component.              |
| `.ods-card-group--minimal`         | Styles applied to the root element when the variant is `minimal`. |
| `.ods-card-group--header`          | Styles applied to the root element when the variant is `header`.  |
| `.ods-card-group__header`          | Styles applied to the header container (title and subtitle).      |
| `.ods-card-group__header--content` | Styles applied to the internal content of the header.             |
| `.ods-card-group__action`          | Styles applied to the action button.                              |
| `.ods-card-group__action--label`   | Styles applied to the label of the action button.                 |

## Usage Examples

### Minimal Variant

The `minimal` variant displays the main content and the action (if provided).

The `header` variant displays only the header (title and subtitle) and the counter, without additional content or action.

## Props

| Property      | Type        | Description                                                                    |
| ------------- | ----------- | ------------------------------------------------------------------------------ |
| `title`       | `string`    | The title displayed in the header.                                             |
| `subtitle`    | `string`    | The subtitle displayed in the header.                                          |
| `variant`     | `string`    | The variant of the component (`minimal` or `header`). Default: `minimal`.      |
| `count`       | `number`    | The value displayed in the counter. If `undefined`, the counter is hidden.     |
| `actionLabel` | `string`    | The label of the action button.                                                |
| `actionClick` | `function`  | Function called when the action button is clicked.                             |
| `children`    | `ReactNode` | Additional content displayed below the header (only in the `minimal` variant). |
