The API documentation of the Link React component. Learn more about the props and the CSS customization points.

## CSS

| Global class         | Description                             |
| -------------------- | --------------------------------------- |
| .ods-lnk             | Styles applied to the root element.     |
| .ods-link\_\_content | Styles applied to the children element. |
| .ods-link\_\_icon    | Styles applied to the icon.             |
| .ods-lnk--primary    | Sets styles of the link.                |
| .ods-lnk--inverse    | Sets styles of the inverse link.        |
| .ods-lnk--neutral    | Sets styles of the neutral link.        |
| .ods-lnk--[sm,md]    | Specifies the font size of the link.    |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Link/Link.tsx) for more detail.
