# Stepper API

The API documentation of the Step React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Stepper } from '@useblu/ocean-react';
```

## Default Usage

<ArgsTable of="." />

## CSS

| Global class                | Description                                                |
| --------------------------- | ---------------------------------------------------------- |
| .ods-sub-header             | Styles applied to the sub-header element.                  |
| .ods-sub-header\_\_title    | Styles applied to the title element.                       |
| .ods-sub-header\_\_subtitle | Styles applied to the subtitle element when it is present. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Input/Input.tsx) for more detail.
