## CSS

| Global class                                | Description                                                |
| ------------------------------------------- | ---------------------------------------------------------- |
| .ods-select-autocomplete\_\_root            | Styles applied to the root element.                        |
| .ods-select-autocomplete\_\_arrow           | Styles applied to the arrow element.                       |
| .ods-select-autocomplete\_\_arrow--up       | Styles applied to the arrow element if the popup is open.  |
| .ods-select-autocomplete\_\_arrow--down     | Styles applied to the arrow element if the popup is close. |
| .ods-select-autocomplete\_\_arrow--disabled | Styles applied to the arrow element if disabled={true}.    |

## Inheritance

The props of the [FormControl](?path=/docs/components-formcontrol--page) component are also available.
