# Inputs

This is the showcase of the Input component of ocean design.

## Base Input

The base input is the base cade of data input, cold be of many types, and have many usages...

### Simple Usage

<Showcase>
  <Input
    label="Text"
    name="text"
    defaultValue="Simple Text Input"
    type="text"
  />
</Showcase>

#### code

```jsx
<Input label="Text" name="text" defaultValue="Simple Text Input" type="text" />
```

## Stepper Input

The stepper input is a diferent kind of input, with only number type, this adds a amount controller for the input, also aviability to set the step of the input using arrow keys.
You could also change the min and max values from stepper:

<Showcase>
  <Stepper
    helperText="Some text here!"
    id="stepper-playground"
    label="Label"
    max={15}
    min={0}
    placeholder="Placeholder..."
  />
</Showcase>

```jsx
<Stepper
  helperText="Some text here!"
  id="stepper-playground"
  label="Label"
  max={15}
  min={0}
  placeholder="Placeholder..."
/>
```
