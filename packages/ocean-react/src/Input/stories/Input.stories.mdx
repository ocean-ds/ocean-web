## CSS

| Global class      | Description                                          |
| ----------------- | ---------------------------------------------------- |
| .ods-input        | Styles applied to the input element.                 |
| .ods-input--error | Styles applied to the input element if error={true}. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/Pagnet/ocean-ds-web/blob/master/packages/ocean-react/src/Input/Input.tsx) for more detail.
