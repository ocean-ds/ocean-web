# Input With Icon API

The API documentation of the Input type With Icon React component.
Learn more about the props and the CSS customization points.

## Import

```javascript
import { Input } from '@useblu/ocean-react';
```

## CSS

| Global class               | Description                                                                                    |
| -------------------------- | ---------------------------------------------------------------------------------------------- |
| .ods-input                 | Styles applied to the input element.                                                           |
| .ods-input--filled         | Styles applied to the input element when it's filled.                                          |
| .ods-input--error          | Styles applied to the input element if error={true}.                                           |
| .ods-input--left           | Styles applied to the input element if position="left".                                        |
| .ods-input--disabled--text | Styles applied to the element if it doesn't have any text and doesn't have an icon as default. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Input/Input.tsx) for more detail.

## Inheritance

The props of the [FormControl](?path=/docs/components-formcontrol--page) component are also available.
