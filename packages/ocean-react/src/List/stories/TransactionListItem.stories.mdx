## CSS

| Global class                                  | Description                                                       |
| --------------------------------------------- | ----------------------------------------------------------------- |
| .ods-transaction-list-item                    | Styles applied to the root element.                               |
| .ods-transaction-list-item\_\_content         | Styles applied to the content element.                            |
| .ods-transaction-list-item\_\_icon            | Styles applied to the icon element.                               |
| .ods-transaction-list-item\_\_information     | Styles applied to the information group container.                |
| .ods-transaction-list-item\_\_level1          | Styles applied to the level 1 type element.                       |
| .ods-transaction-list-item\_\_level2          | Styles applied to the level 2 type element.                       |
| .ods-transaction-list-item\_\_level3          | Styles applied to the level 3 type element.                       |
| .ods-transaction-list-item\_\_level4          | Styles applied to the level 4 type element.                       |
| .ods-transaction-list-item\_\_sub-information | Styles applied to the subinformation in the right side container. |
| .ods-transaction-list-item\_\_value           | Styles applied to the value element.                              |
| .ods-transaction-list-item\_\_tags            | Styles applied to the tags element.                               |
| .ods-transaction-list-item\_\_time            | Styles applied to the time element.                               |
| .ods-transaction-list-item\_\_sub-itens       | Styles applied to the sub itens container.                        |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Tag/Tag.tsx) for more detail.
