# Chip API

The API documentation of the Chip React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Chip } from '@useblu/ocean-react';
```

## Usage

```jsx
import { Chip } from '@useblu/ocean-react';

return (
  <Chip
    label="Chip Name"
    isActive={false}
    counter={1}
    closeAfterAction={false}
   />
```

## CSS

| Global class         | Description                                     |
| -------------------- | ----------------------------------------------- |
| .ods-chip            | Styles applied to the root element.             |
| .ods-chip\_\_content | Styles applied to the element content.          |
| .ods-chip\_\_icon    | Styles applied to the element icon.             |
| .ods-chip\_\_label   | Styles applied to the element label.            |
| .ods-chip\_\_action  | Styles applied to the element action container. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Chips/BasicChip.tsx) for more detail.

export const selectIcon = (icon) => {
  const icons = {
    adjustments: <Adjustments size={16} />,
  };
  return icons[icon] || null;
};
