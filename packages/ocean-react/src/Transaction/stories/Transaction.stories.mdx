# Transaction API

The API documentation of the Transaction React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Transaction } from '@useblu/ocean-react';
```

## Props

<ArgsTable of="." />

The ref is forwarded to the root element.

Any other props supplied will be provided to the root element (native element).

## CSS

| Global class                         | Description                                                       |
| ------------------------------------ | ----------------------------------------------------------------- |
| .ods-transaction                     | Styles applied to the root element.                               |
| .ods-transaction\_\_content          | Styles applied to the content element.                            |
| .ods-transaction\_\_icon             | Styles applied to the icon element.                               |
| .ods-transaction\_\_information      | Styles applied to the information group container.                |
| .ods-transaction\_\_level1           | Styles applied to the level 1 type element.                       |
| .ods-transaction\_\_level2           | Styles applied to the level 2 type element.                       |
| .ods-transaction\_\_level3           | Styles applied to the level 3 type element.                       |
| .ods-transaction\_\_level4           | Styles applied to the level 4 type element.                       |
| .ods-transaction\_\_sub-information  | Styles applied to the subinformation in the right side container. |
| .ods-transaction\_\_value            | Styles applied to the value element.                              |
| .ods-transaction\_\_tags             | Styles applied to the tags element.                               |
| .ods-transaction\_\_time             | Styles applied to the time element.                               |
| .ods-transaction\_\_sub-transactions | Styles applied to the sub transaction container.                  |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Tag/Tag.tsx) for more detail.
