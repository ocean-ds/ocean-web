## CSS

| Global class                 | Description                                          |
| ---------------------------- | ---------------------------------------------------- |
| .ods-container               | Sets a `max-width` at each responsive breakpoint.    |
| .ods-container-fluid         | Sets a `width: 100%` at all breakpoints.             |
| .ods-container-[sm,md,lg,xl] | Sets a `width: 100%` until the specified breakpoint. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/Pagnet/ocean-ds-web/blob/master/src/Grid/Container.tsx) for more detail.
