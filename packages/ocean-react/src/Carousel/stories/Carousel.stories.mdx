## CSS

| Global class               | Description                                                   |
| -------------------------- | ------------------------------------------------------------- |
| .ods-carousel              | Styles applied to the root element.                           |
| .ods-carousel-and-buttons  | Styles applied to the children banners and navigation buttons |
| .ods-carousel-container    | Styles applied to the children banners container              |
| .ods-carousel-item-columns | Styles applied to the children banner                         |
