# Tag APIs

The API documentation of the Tag React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Tag } from '@useblu/ocean-react';
```

## CSS

| Global class            | Description                                                   |
| ----------------------- | ------------------------------------------------------------- |
| .ods-tag                | Styles applied to the root element.                           |
| .ods-tag--default       | Styles applied to the root element with a default type.       |
| .ods-tag--primary       | Styles applied to the root element with a primary type.       |
| .ods-tag--complementary | Styles applied to the root element with a complementary type. |
| .ods-tag--negative      | Styles applied to the root element with a negative type.      |
| .ods-tag--warning       | Styles applied to the root element with a warning type.       |
| .ods-tag--positive      | Styles applied to the root element with a positive type.      |
| .ods-tag\_\_icon        | Styles applied to the icon element.                           |
| .ods-tag\_\_content     | Styles applied to the content element.                        |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Tag/Tag.tsx) for more detail.
