# Snackbar API

The API documentation of the Snackbar React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Snackbar } from '@useblu/ocean-react';
```

export const DinamicSnackbar = (props) => {
  const [open, setOpen] = useState(true);
  const toogle = () => setOpen(!open);
  return (
    <div>
      <Button onClick={toogle}>{open ? 'Close' : 'Open'}</Button>
      <Snackbar id="snackbar-playground" open={open} {...props} />
    </div>
  );
};

## Usage

```jsx
import { Snackbar } from '@useblu/ocean-react';

const [open, setOpen] = useState(false);

const toogle = () => setOpen(!open);

return (
  <Snackbar
    id="snackbar-playground"
    open={open}
    onClose={toogle}
    message="Hello, I'm a snackbar"
  />
```

## CSS

| Global class             | Description                                     |
| ------------------------ | ----------------------------------------------- |
| .ods-snackbar            | Styles applied to the root element.             |
| .ods-snackbar\_\_content | Styles applied to the element content.          |
| .ods-snackbar\_\_icon    | Styles applied to the element icon.             |
| .ods-snackbar\_\_label   | Styles applied to the element label.            |
| .ods-snackbar\_\_action  | Styles applied to the element action container. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Snackbar/Snackbar.tsx) for more detail.
