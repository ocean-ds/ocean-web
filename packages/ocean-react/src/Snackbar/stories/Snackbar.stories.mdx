# Stepper API

The API documentation of the Snackbar React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { Snackbar } from '@useblu/ocean-react';
```

export const DinamicSnackbar = (props) => {
  const [open, setOpen] = useState(false);
  const toogle = () => setOpen(!open);
  return (
    <div>
      <Button onClick={toogle}>{open ? 'Close' : 'Open'}</Button>
      <Snackbar
        id="snackbar-playground"
        open={open}
        onClose={toogle}
        {...props}
      />
    </div>
  );
};

## CSS

| Global class               | Description                                                                                    |
| -------------------------- | ---------------------------------------------------------------------------------------------- |
| .ods-input                 | Styles applied to the input element.                                                           |
| .ods-input--filled         | Styles applied to the input element when it's filled.                                          |
| .ods-input--error          | Styles applied to the input element if error={true}.                                           |
| .ods-input--left           | Styles applied to the input element if position="left".                                        |
| .ods-input--disabled--text | Styles applied to the element if it doesn't have any text and doesn't have an icon as default. |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Stepper/Stepper.tsx) for more detail.
