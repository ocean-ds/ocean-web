## Overview <a name="overview"></a>

The Carousel component in a design system is a dynamic and interactive element that allows for the display
of a series of content items or images in a visually appealing and organized manner. It enables users to
navigate through the content horizontally or vertically, creating an engaging and immersive experience.

## Columns <a name="variants"></a>

The `columns` property of the Carousel component in a design system allows for the
customization of the number of visible slides or items displayed horizontally within
the carousel at any given time.

## Infinite

When the `infinite` property is enabled, the Carousel component cycles through its
slides indefinitely. After reaching the end of the slide sequence, it seamlessly
transitions back to the beginning, creating a loop effect. This means that users can
continuously navigate through the slides in either direction without encountering an
endpoint.

## Autoplay

The `autoplay` property of the Carousel component in a design system enables automatic
and continuous playback of the slides, enhancing the user experience by providing a hands-free
and dynamic browsing experience. The delay time of autoplay is 8s.
