## Overview <a name="overview"></a>

Displays short messages with useful information for a task or something that requires the user's attention.
An alert component is a crucial element in any design system that provides a way to communicate important
messages or notifications to users. It typically consists of a small, highly visible box that appears on top
of other content, and may contain a brief message, a call-to-action, or an icon.

## Usage <a name="usage"></a>

```jsx
import { Alert } from '@useblu/ocean-react';

const MyComponent = () => {
  return <Alert title="Default Title">Hello There!</Alert>;
};
```

## Custom Icon

You can also use the `icon` property to custom svg icon.

Customizing an Alert component icon is a common requirement
when designing a user-friendly and cohesive interface. In many cases,
default icons provided by Alert components may not fit the overall
design aesthetic of a website or application.

## Custom <a name="customization"></a>

CSS provides a powerful toolset for controlling the visual presentation of a website or application,
and knowing how to customize it is crucial for building designs that are consistent, user-friendly,
and responsive. By working closely with designers and stakeholders, developers can create reusable CSS
styles that define the look and feel of various elements, making it easy to maintain a cohesive design system.

<br />

| Global class           | Description                                               |
| ---------------------- | --------------------------------------------------------- |
| .ods-alert             | Styles applied to the root element.                       |
| .ods-alert--has-header | Styles applied to the root element with a title proprity. |
| .ods-alert--error      | Styles applied to the root element with a error type.     |
| .ods-alert--warning    | Styles applied to the root element with a warning type.   |
| .ods-alert--success    | Styles applied to the root element with a success type.   |
| .ods-alert\_\_header   | Styles applied to the header element.                     |
| .ods-alert\_\_icon     | Styles applied to the icon element.                       |
| .ods-alert\_\_title    | Styles applied to the title element.                      |
| .ods-alert\_\_content  | Styles applied to the content element.                    |

If that's not sufficient, you can check the
