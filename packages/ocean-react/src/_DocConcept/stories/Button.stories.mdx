## Overview <a name="overview"></a>

The Button component is a fundamental element within a design system that enables users
to interact with a user interface, triggering actions or submitting forms. It serves as
a key element in enhancing user engagement and facilitating intuitive interactions. The Button
component typically appears as a clickable element with a clearly defined shape, such as
a rectangle or a rounded shape, often accompanied by text or an icon that communicates
its purpose. It is designed to be easily recognizable and distinguishable from other elements
on the screen.

## Blocked

The `blocked` property, also known as the "full-width" property, is an additional customization
option for the Button component in a design system. When enabled, this property allows the
Button to span the entire width of its parent container, disregarding any predefined width
constraints or alignment rules.

## Loading

When the `loading` property is set to true, the Button component displays a loading indicator,
to convey that a process is underway. This is particularly useful when a button triggers an
action that requires some time to complete, such as submitting a form, fetching data, or
performing a server request.

## Disabled

The `disabled` property of the Button component in a design system allows for the deactivation
of a button, preventing user interaction and conveying that a particular action is currently
unavailable or not applicable.

## CSS <a name="customization"></a>

CSS provides a powerful toolset for controlling the visual presentation of a website or application,
and knowing how to customize it is crucial for building designs that are consistent, user-friendly,
and responsive. By working closely with designers and stakeholders, developers can create reusable CSS
styles that define the look and feel of various elements, making it easy to maintain a cohesive design system.

<br />

| Global class               | Description                                             |
| -------------------------- | ------------------------------------------------------- |
| .ods-btn                   | Styles applied to the root element.                     |
| .ods-btn--primary          | Sets primary background and text color of the button.   |
| .ods-btn--primary-critical | Sets styles of the primary critical button.             |
| .ods-btn--secondary        | Sets secondary background and text color of the button. |
| .ods-btn--text             | Sets styles of the text button.                         |
| .ods-btn--text-critical    | Sets styles of the text critical button.                |
| .ods-btn--inverse          | Use the inverse button on dark backgrounds.             |
| .ods-btn--[sm,md,lg]       | Specifies the width of the button.                      |
| .ods-btn--blocked          | Spans the full width of the Button parent.              |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Button/Button.tsx) for more detail.
