## Overview <a name="overview"></a>

The Checkbox component in a design system is a user interface element that allows users to select
or deselect options from a predefined set of choices. It provides a visual representation of a
checkbox accompanied by text, enabling users to indicate their preferences or make multiple selections.
The Checkbox component is widely used in various scenarios, such as forms, settings pages, or configuration
panels, where users need to make multiple selections from a list of options. It provides a clear and
familiar interaction pattern that allows users to indicate their preferences or choose multiple items
simultaneously.

## Indeterminate <a name="variants"></a>

The `indeterminate` property is typically used when dealing with checkboxes that are part of
a group or have hierarchical relationships. It is often applied to a parent checkbox when some,
but not all, of its child checkboxes are selected. This allows users to indicate that a subset
of options within a larger group is chosen, without necessarily selecting all individual checkboxes.

## State with error

The `error` property allows for the display of an error state or message associated with the checkbox.
It provides a visual indication to users when there is an issue or validation error related to the checkbox
selection.
