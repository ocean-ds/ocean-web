## Overview <a name="overview"></a>

A CrossSellCard is a component within a design system that is specifically designed to promote and
encourage cross-selling opportunities for products or services. This component is commonly used in
e-commerce websites, mobile apps, or any platform where multiple offerings are available, and the
goal is to entice users to explore related or complementary items.

## Usage <a name="usage"></a>

```jsx
import { CrossSellCard } from '@useblu/ocean-react';
import emptyImage from './img/empty-image.svg';

const MyComponent = () => {
  return (
    <CrossSellCard
      title="Card Title"
      description="Lorem ipsum dolor sit amet, consectetur adipiscing elit."
      ctaText="Call To Action"
      imageSrc={emptyImage}
      ctaAction={() => console.log('clicked')}
    />
  );
};
```

## Custom image <a name="variants"></a>

The `imageSrc` permit to customize card image. Unlike standard product images, custom images
are often illustrations, graphics, or unique visual representations of the cross-selling items. These
images are carefully designed to catch the user's attention and provide a distinctive visual appeal.

## Custom icon

The `customIcon` are carefully designed to represent the cross-selling items concisely. Custom icons can
range from simple graphical representations to more intricate and stylized symbols, depending on the
design system's aesthetics and goals.
