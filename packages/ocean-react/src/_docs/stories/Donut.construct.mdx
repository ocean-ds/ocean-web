## Overview <a name="overview"></a>

The Donut Chart component in a design system is a graphical representation that displays data in a
circular shape, similar to a pie chart. It is a visually compelling and intuitive way to present
data proportions and comparisons. The Donut Chart component consists of a circular shape divided
into segments, where each segment represents a specific data category or value. The size of each
segment is proportional to the value it represents, allowing users to easily grasp the relative
magnitudes of different data points.

## Usage <a name="usage"></a>

```jsx
import { DoughnutChart } from '@useblu/ocean-react';

const MyComponent = () => {
  const data = [
    {
      id: 1,
      count: 35,
      percentage: 65,
      color: '#13BDBD',
      descriptionLevel1: 'Description level 1',
      descriptionLevel2: 'Description level 2',
    },
    {
      id: 2,
      count: 10,
      percentage: 35,
      color: '#FFA347',
      descriptionLevel1: 'Description level 1',
      descriptionLevel2: 'Description level 2',
    },
  ];

  return (
    <DoughnutChart
      title="Title"
      subtitle="Subtitle"
      centerChartLabel="center label"
      centerChartValue="center value"
      data={data}
    />
  );
};
```
