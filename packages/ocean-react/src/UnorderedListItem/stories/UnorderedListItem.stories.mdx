# Unordered List Item API

The API documentation of the Unordered List Item React component. Learn more about the props and the CSS customization points.

## Import

```javascript
import { UnorderedListItem } from '@useblu/ocean-react';
```

## Props

<ArgsTable of="." />

The ref is forwarded to the root element.

Any other props supplied will be provided to the root element (native element).

## CSS

| Global class                               | Description                                   |
| ------------------------------------------ | --------------------------------------------- |
| .ods-unordered-list-item                   | Styles applied to the root element.           |
| .ods-unordered-list-item\_\_icon           | Styles applied to the icon element.           |
| .ods-unordered-list-item\_\_icon-container | Styles applied to the icon container element. |
| .ods-unordered-list-item\_\_title          | Styles applied to the title element.          |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Tag/Tag.tsx) for more detail.
