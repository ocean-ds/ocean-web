# Design Tokens

Design Tokens são os valores fundamentais que definem a identidade visual do Ocean Design System. Eles garantem consistência em cores, tipografia, espaçamento, sombras e outros elementos visuais em todos os produtos.

## O que são Design Tokens

Design Tokens são pares nome-valor que armazenam decisões de design visual. Eles servem como a "fonte única da verdade" para elementos de design, garantindo consistência e facilitando atualizações em escala.

### Vantagens dos Design Tokens

- **Consistência**: Valores padronizados em todos os produtos
- **Manutenibilidade**: Atualizações centralizadas se propagam automaticamente
- **Colaboração**: Linguagem comum entre design e desenvolvimento
- **Escalabilidade**: Facilita crescimento e evolução do sistema

## Melhores Práticas

### ✅ Faça

- Use sempre tokens em vez de valores hard-coded
- Importe apenas os tokens que você precisa
- Mantenha consistência com a nomenclatura dos tokens
- Use tokens semânticos quando apropriado
- Use as variantes corretas das cores de status:
  - **Pure**: Para textos, ícones e bordas principais
  - **Up**: Para fundos de alertas e cards de status
  - **Down**: Para estados de hover
  - **Deep**: Para estados de pressed/active

### ❌ Evite

- Criar valores personalizados sem necessidade
- Usar valores hard-coded que já existem como tokens
- Modificar valores de tokens diretamente
- Misturar diferentes sistemas de design
- Usar variantes incorretas (ex: usar "Deep" para fundos claros)

## Como Usar

### Importação

```jsx
import {
  colorBrandPrimaryPure,
  spacingInlineXs,
  borderRadiusSm,
  shadowLevel2,
} from '@useblu/ocean-tokens/web/tokens';
```

### Em JavaScript/React

```jsx
// Usando tokens importados
const MyComponent = () => (
  <div
    style={{
      color: colorBrandPrimaryPure,
      padding: spacingInlineXs,
      borderRadius: borderRadiusSm,
      boxShadow: shadowLevel2,
    }}
  >
    Conteúdo
  </div>
);
```

### Em SCSS

```scss
// Importando tokens SCSS
@import '@useblu/ocean-tokens/web/tokens';

.my-component {
  color: $color-brand-primary-pure;
  padding: $spacing-inline-xs;
  border-radius: $border-radius-sm;
  box-shadow: $shadow-level-2;
}
```

### Tokens Disponíveis

#### Cores da Marca

- `colorBrandPrimaryDeep` - Versão mais escura
- `colorBrandPrimaryPure` - Cor principal da marca
- `colorBrandPrimaryDown` - Versão intermediária
- `colorBrandPrimaryUp` - Versão mais clara

#### Cores de Status

##### Status Positive (Sucesso)

- `colorStatusPositiveDeep` - `rgb(45, 169, 79)` - Versão mais escura (pressed)
- `colorStatusPositivePure` - `rgb(61, 204, 100)` - Cor principal para sucesso
- `colorStatusPositiveDown` - `rgb(122, 220, 148)` - Versão intermediária (hover)
- `colorStatusPositiveUp` - `rgb(242, 253, 245)` - Versão muito clara (fundos)

##### Status Negative (Erro)

- `colorStatusNegativeDeep` - `rgb(211, 20, 65)` - Versão mais escura (pressed)
