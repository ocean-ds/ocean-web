# ContextualHero

Componente de destaque utilizado para exibir mensagens personalizadas baseadas na jornada do usuário. Permite combinar título, descrição, lista de itens, botões de ação e uma imagem opcional.

## Importação

```tsx
import { ContextualHero } from '@useblu/ocean-react';
```

### Importação de tipos TypeScript

```tsx
import type {
  ContextualHeroProps,
  ContextualHeroListItemString,
  ContextualHeroAction,
} from '@useblu/ocean-react';
```

## API

### Props

| Prop          | Tipo                                                                     | Padrão | Descrição                                                                                     |
| ------------- | ------------------------------------------------------------------------ | ------ | --------------------------------------------------------------------------------------------- |
| `title`       | `string`                                                                 | -      | **Obrigatório.** Título principal do hero.                                                    |
| `description` | `string`                                                                 | -      | **Obrigatório.** Descrição ou subtítulo do hero.                                              |
| `listItems`   | `(ReactNode \| ContextualHeroListItemString)[]`                          | -      | **Obrigatório.** Lista de itens a serem exibidos.                                             |
| `image`       | `string \| ReactNode`                                                    | -      | Imagem ou componente a ser exibido na lateral direita. Se for string, renderiza como `<img>`. |
| `actions`     | `[ContextualHeroAction] \| [ContextualHeroAction, ContextualHeroAction]` | -      | Botões de ação (máximo 2). O primeiro usa variant `primary`, o segundo usa `tertiary`.        |
| `className`   | `string`                                                                 | -      | Classes CSS adicionais.                                                                       |

### Tipos

#### ContextualHeroListItemString

```tsx
type ContextualHeroListItemString = {
  icon?: ReactNode;
  description: string;
};
```

#### ContextualHeroAction

```tsx
type ContextualHeroAction = {
  label: string;
  onClick: () => void;
};
```

## Estrutura de listItems

O prop `listItems` aceita dois tipos de itens:

### 1. Objeto com descrição (e ícone opcional)

```tsx
listItems={[
  { icon: <CheckCircleOutline size={20} />, description: 'Item com ícone' },
  { description: 'Item sem ícone' },
]}
```

### 2. ReactNode customizado

```tsx
listItems={[
  <ListAction key="1" title="Item 1" description="Descrição" type="text" />,
  <CustomComponent key="2" />,
]}
```

## Acessibilidade

- ✅ Utiliza elementos semânticos HTML
- ✅ Botões de ação são elementos `<button>` nativos
- ✅ Imagem inclui atributo `alt` automaticamente (usando o título)
- ✅ Suporte a navegação por teclado
- ✅ Suporte a ref forwarding

## Boas práticas

- Use títulos claros e concisos
- A descrição deve complementar o título, não repeti-lo
- Limite a lista a 3-5 itens para melhor legibilidade
- Use no máximo 2 botões de ação para evitar sobrecarga visual
- A imagem deve ser relevante ao contexto do hero
